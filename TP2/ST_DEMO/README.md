### `git` vs `github`
* `git`
  * 분산 버전 관리 시스템
  * 컴퓨터 파일의 변경사항을 추적하는 시스템
  * 빠른 수행 속도에 중점을 두고 있는 것이 특징
  * 데이터 무결성, 분산, 비선형 워프플로를 지원
  * 여러 명의 사용자들 간에 해당 파일들의 작업을 조율하기 위한 스냅샷 스트림 기반의 분산 버전 관리 시스템
* `github`
  * 소스코드 관리 + 저장 + 소셜코딩
  * 분산 버전 관리 툴인 git 저장소 호스팅을 지원하는 웹 서비스
  * 오픈 소스를 위한 무상 서비스를 모두 제공
  * 영리적인 서비스도 제공
